# Ejecutar en desarrollo

1. Clonar el repositorio
2. Ejecutar
```
yarn install
```
3. Insatalar Nest CLI 
```
npm i -g @nestjs/cli
```
4. Lavantar la base de datos (Docker)
```
docker-compose up -d
```
## Stack Usado
* Nest
* MongoDB
